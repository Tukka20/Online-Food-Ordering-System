# Online-Food-Ordering-System
A dynamic online food ordering web application built using Java Servlets, HTML, CSS and MySQL with complete cart and order management functionality.
It simulates a real-world online food delivery platform where users can browse restaurants, view menus, add items to cart, and place orders.
